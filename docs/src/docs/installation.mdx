To get started, you need to install our core package (`@devesharp/unform-core`) and one
package based on the client type:

> Unform is compatible with React >= v16.8.0

### React

```bash
yarn add @devesharp/unform-web @devesharp/unform-core
# or
npm i @devesharp/unform-web @devesharp/unform-core
```

### React Native

```bash
yarn add @devesharp/unform-mobile @devesharp/unform-core
# or
npm i @devesharp/unform-mobile @devesharp/unform-core
```
